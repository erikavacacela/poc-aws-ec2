# poc-aws-ec2

## Development

* Clone reposity
* Configure custom ssh keys, origin
```
code .git/config

git@giterika.com:erikavacacela/poc-aws-ec2.git
```

* Configure aws static credentials


### Terraform commands

Init terraform files

```
terraform init
```

Run plan

```
terraform plan
```

Apply changes

```
terraform apply 
```

Remove configuration and resources

```
terraform destroy
```
